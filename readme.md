# Orange CLI
Script para generacion de entornos y direct updates en **Orange/Amena**
### Configuracion
Creamos un archivo .env en la raiz con los siguientes datos:
```
pathRepoOrange=""
pathRepoAmena=""
appIdAppCenter=""
```

### Requisitos
- Appcenter CLI
  npm install -g appcenter-cli


### Iniciar CLI
- **sh init.sh orange**
- **sh init.sh amena**
